# Student Record Manager

A simple **Node.js Command-Line Interface (CLI)** application to manage student records using the File System (`fs`) module. This project performs basic CRUD (Create, Read, Update, Delete) operations and stores data in a JSON file.

## Features

- ➕ Add a new student
- 📋 View all students
- ✏️ Update student details
- ❌ Delete a student
- 💾 Store student records in a JSON file

## Tech Stack

- Node.js
- JavaScript
- File System (`fs`) Module
- JSON

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/lakhokaur2026/Student-Record-Manager.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the application:

```bash
node app.js
```
